💡 First Time? Here's How
1
Get a GitHub token — Go to github.com/settings/tokens/new. Name it "iTabs Publisher", tick the repo checkbox, hit Generate. Copy the token.
2
Paste it above and hit Save Connection. You only do this once — it saves in your browser.
3
Paste any HTML from Claude into the code box, give it a filename, and hit Publish. Your live URL appears in seconds.
4
Share the link — it's live on GitHub Pages at boilermaker-innovator.github.io/itabs-quotes/your-file.html

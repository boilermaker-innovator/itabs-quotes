## What is this?

Free branded price guide widgets that walk customers through their project step by step and give them a ballpark range based on the tradie's actual rates. The customer gets instant pricing. The tradie gets a qualified lead with a full spec sheet.

**The problem:** Most trade websites have a contact form and a phone number. Customers have no idea what things cost, so they bounce or send vague enquiries.

**The fix:** An interactive estimator that educates while it quotes — fence type, height, length, extras — then captures the lead behind a gated breakdown.

## How It Works

```
Customer lands on widget
        ↓
Picks options (type, size, extras)
        ↓
Sees ballpark price guide
        ↓
Enters details → unlocks full breakdown
        ↓
Tradie gets lead email with full spec (no pricing)
```

**Pricing is a sales tool.** The ballpark range keeps customers engaged and filters unrealistic budgets. The actual quote comes from the tradie after a site visit. The lead email never includes pricing — just the clean spec sheet.

## Tech

Each widget is a **single self-contained HTML file**. No build step, no dependencies, no framework. Just HTML + CSS + JS in one file, hosted on GitHub Pages.

- Mobile-first responsive design
- Dark theme with branded accent colours per business
- Educational tooltips (bottom-sheet modals on mobile)
- Step-by-step accordion UI with auto-advance
- Lead capture via [Web3Forms](https://web3forms.com)
- OG meta tags for Facebook/social link previews
- Google Fonts (Plus Jakarta Sans)
